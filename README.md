# Catchy

A side-scrolling catch game built with Java Swing. Play as a monkey hanging from a vine â€” catch bananas for points and dodge bats to survive.

## Prerequisites

- Java 17+
- Maven

## Build and Run

```bash
cd catchy
mvn compile
mvn exec:java -Dexec.mainClass="work.terrencemurray.Main"
```

## Controls

| Key | Action |
|-----|--------|
| W / Up Arrow | Move up |
| S / Down Arrow | Move down |
| F1 | Toggle debug hitboxes |
